# CrowdStrike Internship Task Tracker — Backend

Express/Node API with PostgreSQL for managing internship tasks during my CrowdStrike Sales Excellence Elite internship in Austin, TX (Summer 2025).

## Technologies
- Node.js
- Express
- PostgreSQL
- dotenv
- CORS

## Setup
1. Clone the repo
2. Run `npm install`
3. Create a `.env` file with your DATABASE_URL
4. Run `node index.js`

## API Endpoints
- GET /tasks — get all tasks
- GET /tasks/:id — get a task by ID
- POST /tasks — create a task
- PUT /tasks/:id — update a task
- DELETE /tasks/:id — delete a task
